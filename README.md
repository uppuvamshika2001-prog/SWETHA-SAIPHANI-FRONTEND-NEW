# Swetha SaiPhani Clinics - Healthcare Management System

## About Project
Swetha SaiPhani Clinics is a comprehensive healthcare management system designed to streamline clinical operations. It handles various aspects of hospital management including patient registration, OPD consultations, pharmacy inventory, and billing. The application aims to improve efficiency and provide a seamless experience for both healthcare providers and patients.

## Tech Stack
This project is built using modern web technologies to ensure performance, scalability, and maintainability.

### Frontend Core
-   **React**: UI library for building the interface.
-   **Vite**: Fast build tool and development server.
-   **TypeScript**: Static typing for better code quality and developer experience.

### UI & Styling
-   **Tailwind CSS**: Utility-first CSS framework for styling.
-   **shadcn-ui**: Reusable components built with Radix UI and Tailwind CSS.
-   **Lucide React**: Icon library.
-   **Recharts**: Charting library for data visualization.

### State Management & Data Fetching
-   **React Query (@tanstack/react-query)**: Powerful asynchronous state management.
-   **Axios**: Promise-based HTTP client.

### Forms & Validation
-   **React Hook Form**: Performant, flexible and extensible forms with easy-to-use validation.
-   **Zod**: TypeScript-first schema declaration and validation library.

### Routing
-   **React Router DOM**: Declarative routing for React applications.

### Backend & Database (Integration)
-   **Supabase**: Backend-as-a-Service integration.

## Getting Started

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```
